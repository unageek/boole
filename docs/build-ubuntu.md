# Building on Ubuntu 22.04

## Prerequisites

```bash
sudo apt install clang cmake libcgal-dev libgtest-dev libomp-dev ninja-build zsh
```

## Clone

```
git clone https://github.com/unageek/kigumi.git
```

## Build

```
cd kigumi
./run cmake
./run b
```
