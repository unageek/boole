# Building on Ubuntu 24.04

## Prerequisites

```bash
sudo apt install clang cmake libomp-dev ninja-build
```

## Clone

```
git clone https://github.com/unageek/kigumi.git
```

## Build

```
cd kigumi
./run configure  # or ./run c
./run build      # or ./run b
```

See [Running example programs](run.md).
